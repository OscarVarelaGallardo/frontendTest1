## Installation

```bash
git clone https://github.com/your-username/pruebaFrontEnd.git
cd pruebaFrontEnd
npm install
```

## Usage

Start the development server:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
